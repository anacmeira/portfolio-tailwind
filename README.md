# Portfólio Pessoal - Capacitação byron.solutions

Este projeto consiste no desenvolvimento de um portfólio pessoal responsivo, criado como parte do processo de **Capacitação/Trainee da byron.solutions** de 2026. O objetivo principal foi consolidar conhecimentos em desenvolvimento Front-End, aplicando boas práticas de marcação, estilização moderna e controle de versão com Git.

---

## Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes ferramentas e tecnologias:

*   **HTML5** - Estruturação semântica do conteúdo.
*   **Tailwind CSS** - Framework utilitário para estilização rápida, componentização e design responsivo (Mobile-First).
*   **Git & GitHub** - Controle de versão e gerenciamento de branches (`main` e `develop`).

---

## Funcionalidades e Diferenciais

*   **Design 100% Responsivo:** O layout se adapta perfeitamente desde telas pequenas de smartphones até monitores desktop.
*   **Seção Hero Personalizada:** Apresentação pessoal integrada com links para redes sociais.
*   **Vitrine de Projetos:** Grid estruturada exibindo os últimos projetos desenvolvidos (Câmara Fria, Gestor de Presença, Estudos em Python, entre outros).
